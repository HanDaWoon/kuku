    함수
    
        함수는 프로그램을 작성할때 가장 중요한 개념입니다. 함수는 특정한 기능을 수행하는
    코드의 집합입니다. 함수를 사용하면 코드의 재사용성이 높아지고 유지보수가 쉬워집니다.
    함수는 다음과 같이 선언합니다.
    

      
            def 함수명(매개변수):
                수행할 문장1
                수행할 문장2
                ...
        
    
    함수를 선언할때는 def 키워드를 사용합니다. 함수명은 함수의 이름을 의미합니다.
    매개변수는 함수에 입력으로 전달하는 값을 받는 변수입니다. 수행할 문장은 함수를
    수행할 때 반복적으로 수행되는 문장입니다.
  
    
            def add(a, b):  # a, b는 매개변수
                return a + b
            
            print(add(3, 4))       # 3, 4는 인수
            # 결과: 7
        
    함수를 호출할때 전달하는 입력값을 인수라고 합니다. 함수를 호출하면 함수는 입력값을
    받아서 함수에 정의된 수행문장을 수행합니다. 함수는 수행문장을 모두 수행하면 return문을
    만나거나 함수의 마지막 문장까지 수행하면 함수를 빠져나가게 됩니다. return문은
    함수의 결과값을 돌려주는 역할을 합니다. return문을 사용하지 않으면 함수는 결과값을
    돌려주지 않고 끝납니다. 함수를 호출할때는 함수명 뒤에 괄호를 붙이고 괄호 안에
    입력값을 지정합니다.


    입력값과 결과값에 따른 함수의 형태
    
        입력값과 결과값에 따라 함수는 다음과 같이 4가지 유형이 있습니다.
    

      
            1. 일반적인 함수
            2. 입력값이 없는 함수
            3. 결과값이 없는 함수
            4. 입력값도 결과값도 없는 함수
        
    

      
            1. 일반적인 함수
            def add(a, b):
                result = a + b
                return result

            print(add(3, 4))
            # 결과: 7
        
    

      
            2. 입력값이 없는 함수
            def say():
                return 'Hi'

            print(say())
            # 결과: Hi
        
    

      
            3. 결과값이 없는 함수
            def add(a, b):
                print("%d, %d의 합은 %d입니다." % (a, b, a+b))

            add(3, 4)
            # 결과: 3, 4의 합은 7입니다.
        
    

      
            4. 입력값도 결과값도 없는 함수
            def say():
                print('Hi')

            say()
            # 결과: Hi
